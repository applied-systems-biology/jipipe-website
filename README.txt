 This is a local website and requires some sort of local web server.
 
 Python
 ------
 
 1. Navigate into the docs folder
 2. Open a console
 3. Run: python3 -m http.server 8080
 4. Visit: http://localhost:8080
 
 Chrome
 ------
 
 For Chrome there's an addon: https://chrome.google.com/webstore/detail/web-server-for-chrome/ofhbbkphhbklhfoeikjpcbhemlocgigb
