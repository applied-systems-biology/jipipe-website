    JSON API documentation

    This section is about the JIPipe JSON API that includes the API for creating JSON extensions, but also other data types like a project.

    JSON objects

    The JSON API is object-oriented and re-uses different object types in multiple occasions. You will be referred to the specified documentation.

    • Algorithm node

      • Node in a graph

    • Algorithm type

      • Defines a new algorithm

    • Dependency

      • Project or extension dependency. Refers to an JIPipe extension.

    • Graph Edge

      • Edge in a graph

    • Graph

      • A graph

    • Extension

      • An extension that can be put into the ImageJ plugin folder

    • Metadata

      • Commonly used metadata type in JIPipe.

    • Point

      • A point

    • Project

      • An analysis project

    • Slot definition

      • Defines a data slot
