Detailed Description

template<class Cache>
struct misaxx::misa_cached_data< Cache >

A shared pointer that manages cached data and comes with methods to import and export data from filesystem. Please prefer interacting with cached data using this interface, as it automatically registers caches into the runtime using misa_runtime_base::instance().register_cache.
