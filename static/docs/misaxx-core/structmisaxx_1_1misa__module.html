Detailed Description

template<class ModuleDeclaration>
struct misaxx::misa_module< ModuleDeclaration >

Second part of a MISA++ module. This module dispatcher is responsible dispatching the module-internal workers, setting up the data and submodules.

A MISA module must be always instantiated with a module interface (the root module is instantiated with a default interface)
